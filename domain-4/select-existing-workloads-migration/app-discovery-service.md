# App Discovery Service

## Overview
AWS Application Discovery Service helps you plan migration projects by collecting data about your on-premises servers.

## Key Features
- **Server Discovery**: Automated server inventory
- **Dependency Mapping**: Application dependencies
- **Performance Data**: Resource utilization metrics
- **Usage Analysis**: Application usage patterns

## Use Cases
- Migration planning
- Dependency analysis
- Resource assessment
- Cost estimation

## Benefits
- Automated discovery
- Dependency visibility
- Performance insights
- Migration planning

## Discovery Methods
- **Agent-based**: Detailed data collection
- **Agentless**: Basic discovery
- **Import**: Existing inventory data
- **Partner integration**: Third-party tools

## Integration
- Migration Hub
- Cost Explorer
- Database Migration Service
- Server Migration Service
