# VPC Flow Logs

## Overview
VPC Flow Logs capture information about the IP traffic going to and from network interfaces in your VPC.

## Key Features
- **Traffic Capture**: Network visibility
- **Log Format**: Detailed flow information
- **Storage Options**: S3 and CloudWatch Logs
- **Real-time Monitoring**: Near real-time analysis

## Use Cases
- Network troubleshooting
- Security monitoring
- Compliance auditing
- Cost optimization

## Benefits
- Network visibility
- Security insights
- Compliance support
- Troubleshooting aid

## Log Information
- Source/destination IP
- Source/destination port
- Protocol
- Action (accept/reject)

## Integration
- CloudWatch Logs
- S3 storage
- Lambda processing
- Third-party tools
