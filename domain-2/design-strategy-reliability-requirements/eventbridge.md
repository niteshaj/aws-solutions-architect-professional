# EventBridge

## Overview
Amazon EventBridge is a serverless event bus that makes it easy to connect applications together using data from your own apps, integrated Software-as-a-Service (SaaS) applications, and AWS services.

## Key Features
- Serverless event routing
- Real-time event filtering
- Custom event buses
- Archive and replay capabilities

## Event Sources
- **AWS Services**: Native AWS event integration
- **Custom Applications**: Application-generated events
- **SaaS Partners**: Third-party service integration
- **API Destinations**: External service integration

## Use Cases
- Event-driven architectures
- Application integration
- Real-time data processing
- Workflow automation

## Benefits
- Simplified event routing
- Reduced integration complexity
- Event filtering and transformation
- Cost-effective event processing

## Integration
- Lambda functions
- SQS queues
- Step Functions
- Third-party APIs
