# Compute Optimizer

## Overview
AWS Compute Optimizer is a service that analyzes your compute resources and provides recommendations for optimal configuration to improve performance and reduce costs.

## Key Features
- Automated resource analysis
- Performance optimization recommendations
- Cost optimization suggestions
- Rightsizing recommendations
- Instance type optimization

## Supported Resources

### Compute Resources
- EC2 instances
- Auto Scaling groups
- EBS volumes
- Lambda functions
- ECS services
