# Security Hub & GuardDuty

## Overview
AWS Security Hub and Amazon GuardDuty are integrated security services that provide comprehensive threat detection and security monitoring.

## Amazon GuardDuty

### Features
- Intelligent threat detection
- Continuous monitoring
- Machine learning analysis
- Malware detection
- Network traffic analysis

### Capabilities
- Threat intelligence feeds
- Anomaly detection
- API activity monitoring
- DNS query analysis
- VPC flow log analysis

### Use Cases
- Compromise detection
- Insider threat identification
- Unauthorized access attempts
- Malware and crypto-mining
- Data exfiltration attempts

## AWS Security Hub

### Features
- Centralized security aggregation
- Compliance standards tracking
- Automated security checks
- Security findings management
- Integration with multiple services

### Supported Standards
- CIS AWS Foundations Benchmark
- PCI DSS
- NIST 800-53
- AWS Foundational Security Best Practices
- Custom security standards

### Integrations
- GuardDuty
- Inspector
- Macie
- IAM Access Analyzer
- Config Rules
