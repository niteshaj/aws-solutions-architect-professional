# AWS DMS

## Overview
AWS Database Migration Service helps you migrate databases to AWS quickly and securely.

## Key Features
- **Homogeneous Migration**: Same database engine
- **Heterogeneous Migration**: Different database engines
- **Continuous Replication**: Ongoing sync
- **Minimal Downtime**: Near-zero downtime

## Use Cases
- Database migration
- Database replication
- Development testing
- Disaster recovery

## Benefits
- Minimal downtime
- Data consistency
- Schema conversion
- Continuous replication

## Migration Types
- **Full Load**: Complete data migration
- **CDC**: Change data capture
- **Full Load + CDC**: Combined approach
- **Ongoing Replication**: Continuous sync

## Integration
- RDS databases
- Aurora clusters
- Redshift warehouses
- On-premises databases
