# AWS Backup

## Overview
AWS Backup is a centralized backup service that simplifies backup management across AWS services and hybrid environments.

## Key Features
- Centralized backup management
- Automated backup scheduling
- Cross-region backup
- Cross-account backup
- Policy-based management

## Supported Services
- Amazon EBS
- Amazon RDS
- Amazon DynamoDB
- Amazon EFS
- Amazon FSx
- Storage Gateway
- AWS S3
- VMware workloads

## Core Components

### Backup Vaults
- Logical containers for backups
- Encryption configuration
- Access control policies
- Retention management

### Backup Plans
- Backup schedules
- Backup rules
- Lifecycle management
- Cross-region copy

### Backup Policies
- JSON-based policy definitions
- Resource tagging
- Automated resource discovery
- Compliance enforcement
