# Amazon Detective

## Overview
Amazon Detective makes it easy to analyze, investigate, and quickly identify the root cause of potential security issues or suspicious activities.

## Key Features
- **Graph Analysis**: Relationship mapping
- **Timeline Analysis**: Event correlation
- **Entity Investigation**: Deep dive analysis
- **Automated Analysis**: ML-powered insights

## Use Cases
- Security incident investigation
- Forensic analysis
- Threat hunting
- Compliance investigation

## Benefits
- Faster investigation
- Better visibility
- Automated analysis
- Reduced manual effort

## Data Sources
- CloudTrail logs
- VPC Flow Logs
- DNS logs
- GuardDuty findings

## Integration
- GuardDuty
- Security Hub
- CloudTrail
- VPC Flow Logs
