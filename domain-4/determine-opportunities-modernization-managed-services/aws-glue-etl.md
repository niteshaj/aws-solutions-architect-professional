# AWS Glue & ETL

## Overview
AWS Glue is a serverless data integration service for data discovery, preparation, and ETL processing.

## Key Features
- **Data Catalog**: Central metadata repository
- **ETL Jobs**: Data transformation jobs
- **Crawlers**: Automatic data discovery
- **Job Bookmarks**: Incremental processing

## Use Cases
- Data integration
- ETL processing
- Data warehousing
- Analytics preparation

## Benefits
- Serverless processing
- Automatic scaling
- Cost effective
- Easy integration

## Components
- **Glue Studio**: Visual ETL interface
- **Glue Jobs**: Python/Scala ETL code
- **Data Catalog**: Metadata management
- **Development Endpoints**: Development environment

## Integration
- S3 data lakes
- Redshift warehouses
- RDS databases
- Athena queries

## Data Sources
- S3, RDS, Redshift
- DynamoDB, MongoDB
- JDBC databases
- Streaming data
