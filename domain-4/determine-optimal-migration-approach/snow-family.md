# Snow Family

## Overview
AWS Snow Family provides physical devices to transport petabytes of data into and out of AWS when network connectivity is limited.

## Snow Family Devices
- **Snowball**: Data transfer appliance
- **Snowball Edge**: Compute and storage
- **Snowmobile**: Exabyte-scale data transfer
- **Snowcone**: Portable data transfer

## Use Cases
- Large data transfers
- Network-limited locations
- Edge computing
- Data migration

## Benefits
- Fast data transfer
- Secure transport
- Edge computing
- Cost effective

## Selection Criteria
- **Data Volume**: Snowcone/Snowball/Snowmobile
- **Compute Needs**: Snowball Edge
- **Location**: Remote/edge locations
- **Timeline**: Transfer speed requirements

## Integration
- S3 storage
- Glacier archive
- Edge computing
- Data processing
