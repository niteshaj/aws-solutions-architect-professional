# Savings Plans vs RIs

## Overview
AWS Savings Plans and Reserved Instances (RIs) are both purchasing options that provide significant cost savings compared to On-Demand pricing.

## Reserved Instances (RIs)

### Types of RIs
- **Standard RIs**: Up to 72% savings, flexible changes
- **Convertible RIs**: Up to 54% savings, convertible to other instances
- **Scheduled RIs**: Capacity reservations for specific time windows

### RI Characteristics
- Specific instance type and family
- Specific region or Availability Zone
- 1-year or 3-year terms
- No Upfront, Partial Upfront, All Upfront payment options

### RI Benefits
- Capacity reservation
- Significant cost savings
- Predictable costs
- Exchange flexibility (for Convertible RIs)

## Savings Plans

### Types of Savings Plans
- **Compute Savings Plans**: Up to 66% savings, flexible compute usage
- **EC2 Instance Savings Plans**: Up to 72% savings, EC2 instance flexibility
- **SageMaker Savings Plans**: Up to 64% savings for ML workloads

### Savings Plans Characteristics
- Flexible instance usage
- Region-wide applicability
- 1-year or 3-year terms
- No Upfront, Partial Upfront, All Upfront options

### Savings Plans Benefits
- Instance family flexibility
- Size flexibility
- Region flexibility
- Automatic application

## Comparison Analysis

| Feature | Reserved Instances | Savings Plans |
|---------|-------------------|----------------|
| Flexibility | Limited | High |
| Savings | Up to 72% | Up to 66% |
| Capacity Reservation | Yes | No |
| Instance Family | Fixed | Flexible |
| Region | Fixed | Flexible |
| Exchange Options | Convertible only | Built-in flexibility |
