# Consolidated Billing

## Overview
AWS Consolidated Billing enables organizations to combine payment for multiple AWS accounts through a single management account.

## Key Features
- Single payment method
- Volume pricing benefits
- Cost allocation tags
- Detailed cost reporting
- Free tier aggregation
