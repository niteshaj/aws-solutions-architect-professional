# AWS Config

## Overview
AWS Config provides assessment, auditing, and evaluation of the configurations of your AWS resources.

## Key Features
- **Configuration Tracking**: Continuous monitoring
- **Config Rules**: Compliance evaluation
- **Conformance Packs**: Rule collections
- **Remediation**: Automated fixes

## Use Cases
- Compliance monitoring
- Configuration drift detection
- Security assessment
- Resource inventory

## Benefits
- Continuous compliance
- Automated remediation
- Configuration visibility
- Audit readiness

## Integration
- CloudTrail for logging
- Lambda for custom rules
- SNS for notifications
- CloudWatch for metrics
