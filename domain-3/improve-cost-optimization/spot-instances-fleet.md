# Spot Instances & Fleet

## Overview
EC2 Spot Instances offer spare compute capacity at significant discounts compared to On-Demand prices.

## Key Features
- **Cost Savings**: Up to 90% discount
- **Fleet Management**: Mixed instance strategies
- **Capacity Optimization**: Best available instances
- **Interruption Handling**: Graceful termination

## Use Cases
- Batch processing
- Big data processing
- Container workloads
- Development/testing

## Benefits
- Significant cost savings
- Scalable computing
- Flexible capacity
- High performance

## Strategies
- **Spot Fleet**: Mixed instance types
- **Capacity-Optimized**: Best capacity
- **Lowest Price**: Maximum savings
- **Diversified**: Balanced approach

## Integration
- Auto Scaling
