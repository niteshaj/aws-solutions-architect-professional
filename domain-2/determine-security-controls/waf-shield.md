# WAF & Shield

## Overview
AWS WAF (Web Application Firewall) and AWS Shield provide comprehensive protection for web applications against common web exploits and DDoS attacks.

## AWS WAF Features

### Protection Capabilities
- SQL injection protection
- Cross-site scripting (XSS) prevention
- HTTP flood protection
- Bot control
- Custom rule creation

### Rule Management
- Managed rule groups
- Custom rules
- Rule groups
- Rule priority
- Condition-based rules

### Integration Options
- CloudFront integration
- Application Load Balancer
- API Gateway
- AWS AppSync

## AWS Shield Features

### Shield Standard
- Automatic DDoS protection
- Network and transport layer protection
- Always-on protection
- No additional cost

### Shield Advanced
- Enhanced DDoS protection
- 24/7 AWS DDoS Response Team (DRT)
- Cost protection against DDoS-related usage spikes
- Near real-time visibility
