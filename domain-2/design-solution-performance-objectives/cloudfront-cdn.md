# CloudFront CDN

## Overview
Amazon CloudFront is a fast content delivery network (CDN) service that securely delivers data, videos, applications, and APIs to customers globally.

## Key Features
- Global edge locations
- DDoS protection
- SSL/TLS encryption
- Real-time metrics and logging

## Performance Benefits
- Reduced latency through edge caching
- Improved user experience
- Lower origin server load
- Increased reliability

## Use Cases
- Static and dynamic content delivery
- Video streaming
- API acceleration
- Website acceleration

## Security Features
- AWS WAF integration
- Shield Advanced protection
- Field-level encryption
- Access control

## Integration
- S3 for static content
- EC2 for dynamic content
- Lambda@Edge for customization
- API Gateway for APIs
