# Amazon Macie

## Overview
Amazon Macie is a fully managed data security and data privacy service that uses machine learning and pattern matching to discover and protect your sensitive data stored in Amazon S3.

## Key Features

### Data Discovery
- Automated sensitive data discovery
- Machine learning classification
- Custom data identifiers
- PII detection
