# ACM (TLS/SSL)

## Overview
AWS Certificate Manager (ACM) simplifies the process of provisioning, managing, and deploying public and private SSL/TLS certificates for use with AWS services and internal resources.

## Key Features

### Certificate Management
- Automated certificate renewal
- Free public certificates
- Private certificate authority
- Certificate lifecycle management

### Integration Support

- Elastic Load Balancing
