# 7Rs Migration Strategy

## Overview
The 7Rs migration strategy provides a framework for choosing the right migration approach for each workload.

## The 7Rs Framework
- **Rehost**: Lift and shift to AWS
- **Replatform**: Lift and reshape with optimizations
- **Repurchase**: Move to SaaS solution
- **Refactor**: Re-architect for cloud benefits
- **Rearchitect**: Fundamental changes to architecture
- **Retire**: Decommission unnecessary applications
- **Retain**: Keep in current environment
