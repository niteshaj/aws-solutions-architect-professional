# SNS Fanout

## Overview
Amazon SNS provides fully managed pub/sub messaging for application-to-application (A2A) and application-to-person (A2P) communication.

## Key Features
- High-throughput pub/sub messaging
- Multiple subscription types
- Message filtering
- Cross-region replication

## Subscription Types
- **HTTP/HTTPS**: Webhook delivery
- **Email/Email-JSON**: Email notifications
- **SMS**: Text message delivery
- **SQS**: Queue-based delivery
- **Lambda**: Function invocation

## Use Cases
- Fan-out patterns for notifications
- System alerts and monitoring
- Mobile push notifications
- Event-driven architectures

## Benefits
- Decoupled message delivery
- Multiple delivery channels
- High availability and durability
- Cost-effective messaging

## Integration
- Lambda functions
- SQS queues
- Mobile applications
- Email systems
