# PrivateLink & VPC Endpoints

## Overview
AWS PrivateLink provides private connectivity between VPCs and AWS services without exposing traffic to the public internet.

## Key Features
- Private connectivity to AWS services
- Interface and Gateway endpoints
- Security group controls
- No public IP addresses required
- Integration with VPC

## Types of Endpoints

### Interface Endpoints
- Powered by AWS PrivateLink
- Elastic Network Interface (ENI) based
- Support for most AWS services
- Security group controls
- DNS resolution options

### Gateway Endpoints
- Route table based
- Support for S3 and DynamoDB
- No additional cost
- Simple configuration
- High performance

## Use Cases

- Data privacy and security
